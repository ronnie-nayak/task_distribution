# Task Distribution Research Paper Codes

## Steps to run code
- Create virtual environment
  ```
  python -m venv venv
  ```
- Activate Environment
  - Windows
    ```
    # In cmd.exe
    venv\Scripts\activate.bat
    # In PowerShell
    venv\Scripts\Activate.ps1
    ```
  - Linux
    ```
    source myvenv/bin/activate
    ```
- Run code
  ```
  python3 main.py
  ```
- Deactivate virtual environment
  ```
  deactivate
  ```
